# Asset Inbox

This folder is the download and review area for free assets before they are imported into Unity.

Use it in two steps:

1. Download raw files into the matching category folder here.
2. After review, move only the approved files into `LunarWeekOne_Unity/Assets/...`.

## Folder Map

| Folder | Put Here | Later Unity Target |
|---|---|---|
| `01_Reference` | NASA reference images, moon surface references, station moodboards | Keep outside Unity unless needed for in-game screens |
| `02_Models/Environment` | corridor kits, wall modules, floor pieces, doors | `LunarWeekOne_Unity/Assets/Art/Environment` |
| `02_Models/Props` | consoles, valves, terminals, small station props | `LunarWeekOne_Unity/Assets/Art/Props` |
| `03_Materials/Surfaces` | metal, concrete, moon dust, panel materials | `LunarWeekOne_Unity/Assets/Art/Materials` |
| `03_Materials/Decals` | labels, warning marks, signage, monitor textures | `LunarWeekOne_Unity/Assets/Art/Decals` |
| `04_Audio/Ambience` | station hum, air loop, low drones, moon wind references | `LunarWeekOne_Unity/Assets/Audio/Ambience` |
| `04_Audio/UI` | system beeps, interaction tones, warnings | `LunarWeekOne_Unity/Assets/Audio/UI` |
| `04_Audio/Voice` | temp narration, ritual prompts, archive voice clips | `LunarWeekOne_Unity/Assets/Audio/Voice` |
| `05_Skyboxes` | starfields, earth textures, exterior backdrops | `LunarWeekOne_Unity/Assets/Art/Skyboxes` |

## Intake Rules

- Prefer `CC0` first.
- For Unity Asset Store downloads, keep the original package name in the folder.
- For Freesound, download only clips whose page clearly says `CC0` or a license you have reviewed.
- Keep a note in `asset_manifest.md` for every asset pack you download.
- Do not import everything at once. Import only what supports the next playable slice.
